# 翻牌游戏 - Vue3 + Vite 版本

这是一个使用Vue3、TypeScript和Vite框架重构的翻牌记忆游戏。

## 项目特点

- 使用Vue3组合式API开发
- TypeScript类型支持
- Vite快速开发与构建
- 响应式设计，适配不同难度级别
- 计时器功能

## 游戏说明

玩家可以选择不同难度：
- 简单：6张卡片
- 中等：12张卡片
- 困难：20张卡片

目标是通过翻牌找到所有相同的图片配对。

## 开发与构建

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 打包
npm run build
```

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router

## 原项目说明

本项目是对原有HTML/JavaScript版本翻牌游戏的重构。
